# 🛡️ Data-Driven UI Automation Suite
A Python-based test automation project demonstrating **Parametrization** and **Page Validation** using Pytest and Playwright.

## 🚀 Features
- **Data-Driven:** Uses a centralized `JSON` file to manage multiple test scenarios.
- **Cross-Browser:** Powered by Playwright for fast, reliable execution.
- **Reporting:** Generates HTML visual reports for stakeholders.

## 🛠️ Setup
1. Clone the repo
2. Create venv: `python -m venv venv`
3. Activate: `.\venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac)
4. Install: `pip install -r requirements.txt`
5. Run: `pytest --html=report.html`
